Vue 실무 구분 예제 (검색 결과 리스트 포함)

computed는 계산된 값을 만들고, method는 버튼/이벤트 동작을 실행하며, watch는 값 변화에 반응해 debounce + 검색 결과 리스트를 처리합니다.

이번 버전에서는 watch 예제를 더 실무형으로 바꿨습니다.
검색어를 입력하면 watch가 변화를 감지하고, 잠깐 기다린 뒤 실제 검색을 수행해서 결과 리스트를 화면에 보여줍니다.
computed

계산된 값

화면에 보여줄 파생 데이터를 만들 때 사용합니다. x와 y가 바뀌면 합계가 자동으로 다시 계산됩니다.
자동 계산 결과
{{ sum }}
핵심: 보여줄 계산 결과에 적합
method

동작 실행

버튼 클릭, 저장, 삭제처럼 사용자가 어떤 행동을 했을 때 실행하는 일반 함수입니다.
현재 카운트
{{ count }}
핵심: 버튼 누르면 실행되는 행동에 적합
watch

변화 감지 + 검색

검색어가 바뀌면 watch가 반응하고, debounce 후 검색을 수행합니다. 실제 결과 리스트가 아래에 표시됩니다.
현재 검색어: {{ keyword || '없음' }}
검색 중...
{{ error }}
검색 결과 없음
검색 결과 리스트
  • {{ user.name }}
    {{ user.email }}
아직 결과가 없습니다.
watch 로그
{{ logsText }}
핵심: 값이 바뀌었을 때 뒤에서 할 일을 처리할 때 적합
요약: computed = 계산값 / method = 실행 함수 / watch = 변화 감지 + debounce + 검색 처리
