호이스팅

      변수나 함수 선언이 코드 실행 전에 위로 끌어올려지는 것처럼 동작하는 현상

      1. function 선언문 → 전체 호이스팅
      2. var → 선언만 호이스팅
      3. let / const → TDZ 때문에 접근 불가

함수 vs var 변수

JS는 호이스팅할 때 함수 선언이 var보다 먼저 올라간다.

함수 안에서 호이스팅

let/ const TDZ 문제

      let/const도 호이스팅 되지만 Temporal Dead Zone(TDZ) 때문에 초기화 전에는
      접근이 금지된다. 
      
      - Temporal Dead Zone(TDZ) : let / const 변수에서 초기화되기 전까지 접근할 수 없는 구간
